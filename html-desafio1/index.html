Introdução

(Voltar ao índice)

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ela fornece a estrutura básica para o conteúdo da web, permitindo que os desenvolvedores organizem texto, imagens e outros elementos de forma hierárquica.

O HTML é composto por uma série de elementos, que são representados por "tags". Essas tags são usadas para definir a estrutura e o layout do conteúdo da página.

Exemplo de Tags Básicas

Algumas das tags mais comuns incluem:

  • <h1> a <h6>: Definem títulos de diferentes tamanhos, deste modo:

    1. Isto é um h1

    2. Isto é um h2

    3. Isto é um h3

    4. Isto é um h4

    5. Isto é um h5
    6. Isto é um h6
  • <p>: Define um parágrafo.
  • <small>: Define texto menor.
  • <ol> e <ul>: Definem listas ordenadas e não ordenadas, respectivamente, onde ordenadas são listadas por números e não ordenadas por marcadores.
  • <li>: Define um item de lista.
  • <a>: Define um hyperlink.
  • <hr>: Insere uma linha horizontal.
  • <title>: Define o título da página, que aparece na aba do navegador.
  • <del>: Define texto que foi removido deste modo.
  • <abbr>: Define o significado de uma abreviação ou sigla, como HTML, ao passar o mouse em cima o significado é exibido.
  • <div>: Usada para definir divisões ou seções em um documento, servindo como um contêiner genérico para agrupar elementos. É frequentemente utilizada para estruturar layouts, aplicar estilos CSS ou manipular elementos com JavaScript, especialmente quando não há um elemento semântico mais específico disponível.
  • <span>: Usada para agrupar elementos inline (como texto) para estilização ou manipulação com CSS e JavaScript

Formulários HTML

(Voltar ao índice)

Os formulários HTML são usados para coletar dados do usuário. Eles podem conter uma variedade de elementos, como campos de texto, caixas de seleção e botões de envio.

Algumas tags comuns usadas em formulários incluem:

  • <form>: Engloba todos os elementos do formulário.
  • <input>: Define um campo de entrada. Os atributos mais utilizados são:
    • <input type="checkbox">: Define uma caixa de seleção.
    • <input type="radio">: Define um botão de opção.
    • <input type="text">: Define um campo de texto.
    • <input type="password">: Define um campo de senha, onde os caracteres são ocultos.
    • <input type="email">: Define um campo de entrada de e-mail.
    • <input type="number">: Define um campo de entrada numérica.
    • <input type="date">: Define um campo de entrada de data.
    • <input type="file">: Define um campo para upload de arquivos.
  • <label>: Define um rótulo para um elemento de formulário.
  • <textarea>: Define uma área de texto.
  • <select>: Define uma lista suspensa.
  • <button>: Define um botão.

Formatação HTML

(Voltar ao índice)

A Formatação HTML inclui as seguintes tags:

  • <mark>: Destaca texto, como este texto destacado.
  • <i>: Define texto em itálico, como este texto em itálico.
  • <u>: Define texto sublinhado, como este texto sublinhado.
  • <strong>: Define texto em negrito, como este texto em negrito.
  • <sub> e <sup>: Definem texto subescrito e sobrescrito, respectivamente, deste modo:
    • Isto é um sub
    • Isto é um sup
  • <blockquote>: Define uma citação em bloco.

    • Esta é uma citação em bloco.

  • <font>: Define a fonte ou a cor do texto (embora seja obsoleta em HTML5, ainda é usada em alguns casos).
  • <fieldset>: Agrupa elementos de formulário por seção:
    Título da Seção Deste modo

Tags de Mídia

(Voltar ao índice)

As tags de mídia em HTML são usadas para incorporar conteúdo multimídia, como imagens, vídeos e áudio. Algumas das principais tags de mídia incluem:

  • <img>: Usada para incorporar imagens.
  • <video>: Usada para incorporar vídeos.
  • <audio>: Usada para incorporar áudio.
  • <iframe>: Define um quadro inline, que serve para incorporar conteúdos externos, como páginas HTML, vídeos do YouTube e mapas do Google.

Tabelas HTML

(Voltar ao índice)

As tabelas são usadas para organizar dados em linhas e colunas. As principais tags utilizadads numa tabela HTML são as seguintes:

  • <table>: Engloba os elementos de uma tabela.
  • <thead>: Agrupa o cabeçalho da tabela.
  • <tbody>: Agrupa o corpo da tabela.
  • <tfoot>: Agrupa o rodapé da tabela.
  • <tr>: Define uma linha na tabela.
  • <th>: Define um cabeçalho de coluna.
  • <td>: Define uma célula da tabela.
  • <caption>: Define um título para a tabela.
Exemplo de uma tabela:
Cabeçalho 1 Cabeçalho 2
Dado 1 Dado 2

HTML Semântico

(Voltar ao índice)

O HTML semântico é uma abordagem que utiliza tags HTML para descrever o significado do conteúdo, em vez de apenas sua aparência. Isso melhora a acessibilidade e a SEO (Search Engine Optimization), ou seja, contribui para otimizar o posicionamento do site em mecanismos de buscas.

Algumas tags semânticas incluem:

  • <header>: Define o cabeçalho de um documento ou seção.
  • <main>: Define o conteúdo principal de um documento.
  • <footer>: Define o rodapé de um documento ou seção.
  • <figure>: Define conteúdo ilustrativo, como imagens, gráficos ou diagramas, onde podemos adicionar um figcaption que é um texto relacionado à imagem.
  • <aside>: Define conteúdo relacionado, mas não essencial, ao conteúdo principal.
  • <nav>: Define uma seção de navegação, geralmente composta por uma lista nao ordenada.
  • <article>: Define um conteúdo independente e autocontido.
  • <section>: Define uma seção genérica de um documento.
